Jaune - Zone incertaine selon Dirichlet
La carte d'incertitude met en évidence les pixels où la prédiction est moins fiable, à partir d'une estimation Dirichlet calculée sur les probabilités du modèle.
Le jaune marque les pixels où le modèle hésite (probabilité proche de 0.5), cohérent avec les zones de transition ambiguës identifiées par les experts (Martin-Abadal et al., 2018).
